JS/HTML ads block manager. Allows you to create and insert blocks of code anywhere on the blog.

== Description ==

This is an Ads management Wordpress plugin. It offers unlimited count of code blocks, which can be easily customized and placed wherever you want.

Main **features**:
1. Unlimited ads/widgets creation
2. A lot of default places for blocks (Top/Middle/Bottom/After some tags/After Post)
3. Simple native interface
4. Support any Ads code (HTML/JS)
5. Deffered js ads initialization in footer
6. Include/Exclude ID policy
7. Group ads for faster management
8. Import from other ads management plugins

== Installation ==

1. Unzip the downloaded plugin and upload the folder "rp-ads-manager" to the "/wp-content/plugins/" directory.
2. Go to the plugin menu in WordPress and activate the plugin.
3. Go to the "Ads Manager" page and begin create ads.
